# 一. 问题及背景

​	斗地主是一项经典的三人游戏，玩家根据身份和规则打出手中的牌，最终牌出完的一方获胜。本项目主要实现在线斗地主，能够允许玩家在客户端上与其他人同台竞技，软件用户界面良好、操作简单、功能明确。
